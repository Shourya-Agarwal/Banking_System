# Banking_System
This banking system was developed in scope of making transactions smooth and fast between customers. It helps in tranfers credits between users.
Here the user can perform transactions between the customers registered in our bank. The transactions are stored in the sqllite database. 
And the transaction summary are instantly displayed.
Also, the amount of each of the users involved in the transaction are updated.
It is developed using html, css, JS and Django framework of python. And the database used is SQLlite.
